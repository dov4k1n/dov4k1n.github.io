О сайте

На сайте не используется javascript. Страницы являются статическими html файлами со стилями css. Во время разработки они генерируются из markdown и файлов с html-заготовками с помощью скриптов на языке shell, утилит GNU и программы smu (Simple MarkUp). Весь исходный код можно посмотреть по ссылке https://github.com/dov4k1n/dov4k1n.github.io.

Шрифты не настроены намеренно. Вам должен показываться шрифт по умолчанию из настроек вашего браузера. Вы можете изменить его на любой другой, который вам нравится больше. Таким образом, загрузка страниц не замедляется и вы можете читать сайт со своим любимым шрифтом.
